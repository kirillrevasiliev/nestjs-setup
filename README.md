## Installation

```bash
$ npm install
```

## Copy .env
```bash
$ cp .env.example .env
```

## Run docker container with postgres
**(or install and configure yourself)**
```bash
$ docker-compose up
```
## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
